## Installation

```
$ git clone https://github.com/potato4d/nuxt-and-wp-api-on-docker.git
$ cd nuxt-and-wp-api-on-docker
$ cd ./nuxt; cd yarn; cd ../
$ chmod +x ./wordpress/cmd.sh
$ docker-compose up
```
